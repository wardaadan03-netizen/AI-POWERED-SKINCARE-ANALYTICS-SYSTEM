# 🧴 AI-Powered Skincare Analytics System

## 📌 Overview

The **AI-Powered Skincare Analytics System** is a machine learning-based application designed to analyze skin-related data and provide personalized skincare insights and recommendations. The system leverages data science and AI techniques to predict skin types/concerns and suggest suitable skincare routines.

---

## 🎯 Problem Statement

People often struggle to choose the right skincare products due to lack of personalized guidance. This project aims to solve this problem by using machine learning to analyze skin attributes and recommend appropriate skincare solutions based on data-driven insights.

---

## 🚀 Key Features

* Skin type / condition prediction using ML models
* Personalized skincare recommendations
* Data preprocessing and feature engineering pipeline
* Exploratory Data Analysis (EDA) for insights
* Scalable ML workflow
* User-friendly prediction output

## ⚙️ How to Run This Project

### 1. Clone the repository


### 2. Install dependencies


```bash
python app.py
```

